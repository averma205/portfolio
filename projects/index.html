SDSU Big Data Hackathon

Our team won second place in the 2024 Big Data Hackathon, sponsored by Sharp HealthCare, whose theme was “Enhancing Healthcare’s Digital Front Door”. We designed a working app prototype which aimed to build healthy habits in children from marginalized communities which could be integrated into the existing Sharp HealthCare app.

Meteorite fall analysis

Analyzed 1,000 meteorite landings using Python to assess meteorite mass and landing area distribution.

UCSD Synthesis 1 Project

Used a combination of a sentiment analysis algorithm and aid from native speakers to determine the change in tone that occurs when a sentence is translated using computer translation technology.

Data Analysis of Taylor Swift’s Discography

Analyzed trends using 20+ audio features provided by Spotify to build interactive search and song recommendation tools, such as a tool to determine track similarity.
